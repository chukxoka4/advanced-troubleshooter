# advanced-troubleshooter

A codebase-aware support assistant for product support teams: agents ask questions in plain English, the assistant answers grounded in the live source code of the product they support, and — when a genuine bug or feature request surfaces — it drafts a structured GitHub issue the agent can review and file in one click.

See [`architecture-plan-codebase-assistant.md`](./architecture-plan-codebase-assistant.md) for the full design and [`EXECUTION.md`](./EXECUTION.md) for the commit-by-commit build plan. Team-facing documentation will live in `docs/` once it is written.
